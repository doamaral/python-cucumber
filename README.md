# Installation
- Run `source venv/bin/activate`
- Run `pip install -r requirements.txt`
- It's using Firefox, so make sure you set Firefox Binary Path on `config.py` set `FIREFOX_EXECUTABLE_PATH`
- To run tests: `behave`
